## 🧠 Recommendation Engine

### 1️⃣ Skill Collection

The user provides 3–5 technical skills.

### 2️⃣ TF-IDF Vectorization

Converts skills into weighted vectors where rare and specialized skills receive higher importance.

### 3️⃣ Cosine Similarity

Measures similarity between user skills and job-role skill profiles.

### 4️⃣ Ranking

Roles are ranked based on similarity scores.

### 5️⃣ Recommendation

Top matching career paths are displayed with explanations and learning suggestions.
